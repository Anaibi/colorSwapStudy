[h2] Text settings:

[b]Text with mostly browser default font-size and font-family, the only defined values (can be changed in _settings.scss) are:

  • body font-size: 16px. Change [code]$font_ratio: 16px;.
  • body font-family: sans-serif. Change $font_family_1: sans-serif;.
  • Stripes font-family: serif. Change $font_family_2: serif;.

Links have hover bold weight and on hover switch colors: [a]a link.

[em]Emphasized text is as default for browser.
